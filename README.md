## Deskripsi Program
Program ini adalah program Python sederhana untuk membuat struk pembayaran swalayan.

Program akan menghitung total pembayaran secara otomatis berdasarkan harga barang dan jumlah beli.

## Input Program
* ID Member
* Nama Barang
* Harga Barang
* Jumlah Beli
* Uang Bayar

## Proses Perhitungan
* Total = Harga Barang x Jumlah Beli
* Diskon = 10% jika menggunakan ID Member
* Sub Total = Total - Diskon
* PPN = 10% dari Sub Total
* Total Bayar = Sub Total + PPN
* Kembalian = Uang Bayar - Total Bayar

## Output Program
Program akan menampilkan struk pembelian yang berisi:
* Nama barang
* Harga barang
* Jumlah beli
* Total
* Diskon
* Sub total
* PPN
* Total bayar
* Uang bayar
* Kembalian
