# Tugas Quiz Python Pembayaran

Nama: Kanaya Tiara Aditri  
Kelas: 2-SIKA-1  

## Deskripsi Program
Program ini digunakan untuk menghitung pembayaran swalayan secara otomatis.

Data yang diinput:
- ID Member
- Nama Barang
- Harga Barang
- Jumlah Beli

Data yang dihitung otomatis:
- Total
- Diskon
- Sub Total
- PPN
- Total Bayar
